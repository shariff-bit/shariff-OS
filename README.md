# Shariff OS

Shariff OS is an AI-powered personal operating system built for one person: Shariff.

It is not a generic habit tracker or todo app. It is designed to convert long-term missions into measurable daily execution.

The AI acts as Shariff's personal COO.

## Core Missions

- Mission Alpha: Career
- Mission Beta: OpenMed Labs
- Mission Gamma: DSA Champ
- Mission Delta: Personal Life

Every meaningful task, review, and recommendation should connect to one of these missions.

## MVP Direction

The MVP is Android-first Flutter with Supabase as the backend.

The first product goal is to make Shariff OS useful for:

- morning planning
- daily mission execution
- evening review
- journaling
- Life Score tracking
- AI COO recommendations
