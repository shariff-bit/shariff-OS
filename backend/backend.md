# Backend

This folder will contain backend-related project files.

Planned backend direction:

- Supabase Auth
- Supabase Postgres
- Supabase Storage
- Supabase Edge Functions where needed
- AI calls kept behind backend functions when secrets are involved

The first backend milestone is to create the Supabase schema for:

- missions
- tasks
- daily plans
- evening reviews
- journals
- mood entries
- Life Score entries
- Android usage entries

See `docs/Database.md` for the current database plan.
